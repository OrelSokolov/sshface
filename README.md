# Sshface

This is simple ssh identity changer.

1. Move .ssh to .ssh-profile1 or smth like this
2. Create new .ssh profile and move to .ssh-profile2
3. run sshface to choose between, it will create symlink from this list

## Installation

    $ gem install sshface

## Usage

    sshface
